Отчет по проекту: Task Master (React + TypeScript)
1. Описание проекта
Приложение представляет собой современный менеджер задач (To-Do List), созданный на базе React и Vite. Основная цель — управление личными задачами с возможностью фильтрации и сохранения данных.

2. Технологический стек
Framework: React 18 (с использованием хуков useState, useEffect)

Language: TypeScript (для строгой типизации данных)

Build Tool: Vite (для быстрой сборки и HMR)

Storage: LocalStorage API (сохранение данных в браузере)

Styling: CSS3 (Flexbox, Grid, Animations)

3. Реализованный функционал
Добавление задач: Мгновенное создание новой задачи через поле ввода.

Удаление: Возможность удалять отдельные задачи или очищать весь список.

Статус выполнения: Интерактивные чекбоксы для переключения состояния "Готово".

Фильтрация: Три режима просмотра: "Все", "В работе", "Завершенные".

Сохранение: Приложение автоматически запоминает ваши задачи после перезагрузки страницы.
